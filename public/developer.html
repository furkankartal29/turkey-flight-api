Arama Motoruna Dön

Veri Şeması ve API Özellikleri

Arama motorumuzun sunduğu API uç noktaları, Türkiye genelindeki tarifeli uçuşları standartlaştırılmış veri formatıyla sunar. Entegrasyonlarınızı kolaylaştırmak adına durum kodları ve evrensel zaman formatları (UTC) kullanılmaktadır.

Uçuş Durum Kodları (statusCode / statusText)

Metinsel Türkçe durumlar yerine, API entegrasyonlarınızda aşağıdaki standart kodları kullanabilirsiniz:

Kod Metin Açıklama
1 SCHEDULED Uçuş tarifeli zamanında planlandı
2 DELAYED Uçuşta gecikme (rötar) mevcut
3 BOARDING Uçağa biniş başladı, kapıya gidin veya son çağrı
4 LANDED Uçak varış havalimanına indi
5 CANCELLED Sefer iptal edildi
0 UNKNOWN Bilinmeyen durum veya pano verisi eksik

Evrensel Zaman (UTC) ve Tahmin Modeli

🌍 Zaman Dilimi Dönüşümü:
Türkiye yerel saati sabit olarak UTC+3 (TRT)'dir. API'den dönen *Utc son ekli alanlar, bu yerel saatlerden 3 saat geri alınarak standart ISO 8601 UTC formatına (örn: 2026-07-08T05:00:00.000Z) dönüştürülmüştür.
⚙️ Tarihçeli Kendi Kendini İyileştirme (Self-Healing):
Kalkış veya varış panosundan çekilemeyen eksik veriler, veritabanındaki geçmiş uçuş süreleri veya rota tarihçesi analiz edilerek otomatik tamamlanır. Tamamlanan bu kayıtlar API çıktısında isEstimatedArrival: true veya isEstimatedDeparture: true flag'leri ile işaretlenir ve sistem tahmini olduğunu belirtir.

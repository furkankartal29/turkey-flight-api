Geliştirici REST API

Arama motorumuzun kullandığı uç noktaları sorgulayarak doğrudan rota tabanlı (kalkış ve varış parametreli) JSON listelerini çekebilirsiniz.

GET /api/flights?departure=SAW&arrival=ADB Kalkış-Varış Rota API'si
GET /api/flights/search Uçuş Kodu & Tarih API'si
GET /api/airports Havalimanları Sözlüğü
